# FPS Tester

Simple tool to check if your PC is fast enough for gaming.

## What it does

Runs stress tests to see your real FPS, tells you what games you can play.

## How to run

### Python version
```bash
pip install pygame psutil
python fps_tester.py
```

### Browser version
Just open `browser-tester.html` in your browser. Works offline.

## Tests

13 GPU tests (particles, polygons, fractals, etc) and 4 system tests (CPU, RAM, Disk, Monitor).

## Controls

- Numbers/Letters to pick a test
- 
- E also exits (in web version)
- S for settings (Python only)

## Keys during test

| Key | What |
|-----|------||
| E | Exit (web) |
| S | Settings |

## What you get

- Real FPS counter
- Min/Max/Avg FPS
- Bottleneck info
- Game recommendations
