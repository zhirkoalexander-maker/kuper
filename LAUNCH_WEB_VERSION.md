## 🎯 Как Пользоваться?

### 1️⃣ Запуск теста
- Нажать "Tests" в меню
- Выбрать один из 6 тестов
- Тест автоматически запустится на 30 сек

### 2️⃣ Во время теста
- Видеть real-time FPS (обновляется каждую сек)
- Видеть Average, Min, Max FPS
- Красивая анимация в зависимости от теста

### 3️⃣ Результаты
- **Performance Score 1-10** - общая оценка
- **Bottleneck Detection** - что мешает
- **Два графика** - FPS тренд и метрики
- **Рекомендации** - что делать

### 4️⃣ История
- Нажать "History"
- Видеть все прошлые тесты
- Удалить отдельный тест или все

## 💡 Примеры Использования

### Сценарий 1: Новый Компьютер
1. Откроешь веб-версию
2. Запустишь все 6 тестов по очереди
3. Посмотришь результаты (скорее всего 8-10)
4. Сохраняются в истории автоматически
5. Можешь сравнить результаты со временем

### Сценарий 2: Проблемы с FPS
1. Запустишь тест
2. Получишь score 4/10
3. Прочитаешь рекомендации:
   - "⚠ High FPS fluctuations"
   - "! Upgrade CPU or GPU"
4. Сохраняется в истории
5. После апгрейда запустишь снова и сравнишь

### Сценарий 3: Отслеживание Прогресса
1. Запустишь тесты каждую неделю
2. История сохраняет все результаты
3. Видишь прогресс (или падение FPS)
4. Можешь найти причину проблемы

## 🔧 Технические Детали

### Стек Технологий
```
Frontend:
├── HTML5 (структура)
├── CSS3 (красивый дизайн)
├── JavaScript (логика)
└── Chart.js (графики)

Graphics:
├── Canvas API (рисование)
├── RequestAnimationFrame (60 FPS)
└── SVG возможности

Storage:
├── localStorage (история)
└── JSON (сохранение)
```

### Тесты работают так:
1. Canvas получает контекст 2D
2. RequestAnimationFrame вызывает draw function
3. Каждую секунду считается FPS
4. После 30 сек автоматически стоп
5. Результаты сохраняются
