FPS Tester

See what your computer can actually do

What

FPS Tester runs GPU and CPU stress tests. Real numbers. Real results. Tells you which games you can actually play on your hardware.

Install

git clone https://github.com/zhirkoalexander-maker/kuper.git cd kuper && python -m venv .venv source .venv/bin/activate pip install pygame psutil python fps_tester.py

Usage

  • Pick a test (13 GPU + 4 System)
  • Watch FPS in the corner
  • Press ESC when done
  • Get game recommendations
