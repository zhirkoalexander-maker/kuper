📊 What is FPS Tester?

FPS Tester is a comprehensive performance analysis tool designed to stress test your GPU and CPU. It provides real-time metrics, system monitoring, and gives you insights into what games and applications your computer can handle.

With 13 interactive GPU tests, 4 system monitoring tools, and intelligent recommendations, you'll know exactly what your hardware can do.

✨ Key Features

🎨 13 GPU Tests

From particle storms to fractal rendering - comprehensive GPU stress testing with visual effects

⚡ 4 System Tests

CPU, RAM, Disk I/O, and System Monitor for complete performance analysis

📈 Real-time Metrics

Live FPS display, system stats, and performance HUD during tests

🎮 Game Recommendations

Based on your test results, see which games you can play at maximum settings

🛡️ Crash Protection

Silent protection - if your system can't handle it, it tells you directly

🚀 Installation & Usage

Requirements

Python
3.9+
pygame
2.6.1+
psutil
5.0+
OS
Windows, macOS, Linux

Quick Start

1. Clone the repository:

git clone https://github.com/zhirkoalexander-maker/kuper.git
cd fps-tester

2. Create virtual environment:

python -m venv .venv
source .venv/bin/activate # On Windows: .venv\Scripts\activate

3. Install dependencies:

pip install pygame psutil

4. Run the application:

python fps_tester.py

📖 How It Works

Testing Process

  • Select test category (FPS or System)
  • Choose specific test (13 GPU or 4 System tests)
  • Difficulty automatically set to MEDIUM level
  • Real-time performance metrics display in top-left corner
  • Controls display in top-right corner (ESC to exit, S for settings)
  • Test completes when you press ESC
  • Receive detailed results with game recommendations
