### 2. Performance Score
```

- Базовая оценка: 10 баллов
- FPS < 30: -4 балла
- FPS 30-60: -2 балла
- FPS 60-120: -0.5 балла
- FPS >= 120: 0 баллов

Визуализация:
🟢 9-10  = Excellent
🟡 7-8   = Very Good
🟠 5-6   = Good
🔴 3-4   = Fair
🔴 1-2   = Critical
```

```

- >= 120: Все игры на max
- 90-120: AAA игры на High 1440p
- 60-90:  Casual gaming 1080p
- 45-60:  Lower settings needed
- 30-45:  Minimum settings only
- < 30:   Hardware upgrade needed
```

## 🎓 Как Это Работает?

### Процесс Теста

```
1. Пользователь нажимает "Start Test"
2. Открывается canvas элемент
3. RequestAnimationFrame вызывает draw function
4. Функция рисует 60 фреймов в секунду
5. Каждую секунду подсчитывается FPS
6. После 30 секунд тест останавливается
7. Результаты обрабатываются
8. Показываются графики и рекомендации
9. Данные сохраняются в localStorage
```

### Алгоритм Оценки

```javascript
function calculateScore(avgFps) {
    let score = 10;
    
    if (avgFps < 30) score -= 4;
    else if (avgFps < 60) score -= 2;
    else if (avgFps < 120) score -= 0.5;
    
    return Math.max(1, Math.min(10, score));
}
```

### График Сохранения

```javascript
const history = JSON.parse(
    localStorage.getItem('fpsTestHistory') || '[]'
);

// Содержит последние 50 результатов:
[{
    test: 'particle-storm',
    score: 8.7,
    avgFps: 87,
    date: 'Feb 27, 2026, 5:30 PM',
    timestamp: 1709000000000
}, ...]
```
