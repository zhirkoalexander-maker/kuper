## ⚠️ Safety Guidelines

### EXTREME Tests Warning
These tests are for enthusiasts who want to find their system's limit:

1. **Close all important applications first**
2. **Monitor system temperature** (don't let it exceed 90°C)
3. **Have force-restart ready** (Cmd+Option+Esc on Mac)
4. **Run on AC power**, not battery
5. **Don't run overnight**
6. **System WILL freeze/crash** - this is expected

### If Your System Crashes
- Force restart computer (Cmd+Option+Esc)
- Wait 30 minutes before running another test
- Run EASY tests only in future
- Check your system temperature
- Consider undervolting your GPU

### Temperature Warnings
- Safe: < 75°C
- Caution: 75-85°C
- Warning: 85-95°C
- Danger: > 95°C (stop test immediately)

## 🎓 Understanding Bottlenecks

### GPU Limited
- FPS: >120
- CPU Usage: <60%
- RAM Usage: <70%
- **Solution:** GPU is maxed out, upgrade graphics card

### CPU Limited
- FPS: <90
- CPU Usage: >80%
- RAM Usage: <70%
- **Solution:** CPU is maxed out, upgrade processor

### RAM Limited
- FPS: Unstable/dropping
- RAM Usage: >85%
- CPU Usage: Variable
- **Solution:** Add more RAM (minimum 16GB recommended)

### Balanced
- FPS: 60-120
- CPU Usage: 40-70%
- RAM Usage: 50-70%
- **Solution:** No urgent upgrade needed
